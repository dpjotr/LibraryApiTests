Тест-кейсы для тестирования Rest API приложение - библиотека книг

1.  Предусловия
    1.1 Предусловие 1:
        - Api доступен по ссылке http://localhost:5000/api/books
        - Данные о книгах находятся в изначальном состоянии, т.е. содержат следующие книги:        
        {
            "books": [
                {
                    "author": "Роберт Мартин",
                    "id": 1,
                    "isElectronicBook": false,
                    "name": "Чистый код",
                    "year": 1998
                },
                {
                    "author": "Томас Кормен, Чарльз Лейзерсон",
                    "id": 2,
                    "isElectronicBook": true,
                    "name": "Алгоритмы: построение и анализ",
                    "year": 1989
                }
            ]
        }
